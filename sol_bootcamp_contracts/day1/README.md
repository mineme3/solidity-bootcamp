# Solidity Bootcamp - Day 1 Task

## SimpleStorage Contract
A basic smart contract that stores and retrieves a number on the blockchain.

### Features
- `store(uint256)`: Save a number to the blockchain
- `retrieve()`: View the stored number
- `increment()`: Increase the stored number by 1 (Bonus)

### Usage
1. Deploy the contract in Remix IDE
2. Call `store(10)` to save a value
3. Use `retrieve()` to view the current value
4. Call `increment()` to increase the value by 1

### Testing
- Store value: 10
- After increment: 11
- Retrieve returns: 11
