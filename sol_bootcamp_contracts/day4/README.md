# Bank Contract with Foundry Testing

A simple ETH bank contract with full Foundry test coverage.

## How to Run Tests

1. Install Foundry: https://book.getfoundry.sh/getting-started/installation
```
curl -L https://foundry.paradigm.xyz | zsh
source ~/zshenv
#then run foundryup 
#verify installation 
forge --version 
```
2. Clone this repo and go to day4 task folder
3. Run:
   ```zsh
   forge build
   forge test -vv   
```
##Notice I'm using zsh shell you can use your shell type by changing zsh to your shell name type.
